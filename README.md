## 🌟 Overview

**TypeBlitz** is a full-stack, production-grade typing certification and training platform specifically designed to meet the demands of two competitive fields:
1. **Indian Government Exam Aspirants** preparing for typing tests in **SSC CGL / CHSL, UPSC Civil Services, Banking (IBPS/SBI), Railways (RRB), and Police exams**.
2. **Software Developers & Competitive Typists** aiming to speed up coding workflows by practicing symbols, code keywords, brackets, and real-world script segments.

By bridging arcade gamification with strict, high-fidelity metrics, TypeBlitz guides users from home-row fundamentals to speed-mastery benchmarks.

---

## ✨ Key Features

### 🔒 Strict WPM & Accuracy Filtering
Unlike generic typing sites that save failed attempts, TypeBlitz implements a **strict 90% accuracy threshold** for high scores. 
- Sessions with **accuracy below 90%** are marked as failed, meaning they **do not** qualify for your personal best, do not advance your levels, and do not appear on the Global Leaderboards.
- This prevents keyspamming and ensures that leaderboard rankings reflect real, high-accuracy typing capability.

### 🎯 Strict Mode (Block on Error)
Toggle **Strict Mode** in the typing arena to instantly block incorrect keystrokes.
- The input indicator will turn red and freeze when a typo is typed.
- You must immediately backspace and type the correct character to proceed.
- Perfect for building absolute muscle memory and mimicking strict government exam typing guidelines.

### 📊 Global Leaderboard & Rank Tracking
- Tracks your **Global Rank** (e.g., `#3`) dynamically across the platform.
- Leaderboard scores are verified server-side.
- Filters by game mode so you can view your standing on individual games or overall.

## 🛠 Tech Stack & Folder Reorganization

TypeBlitz is structured as a clean, modular pnpm monorepo separating code logic, schema specifications, and client generation.

```
typeblitz/ (Monorepo Root)
├── frontend/                     # React 19 + Vite 7 App (replaces artifacts/typeblitz)
│   ├── src/
│   │   ├── components/           # Custom reusable layout & game elements
│   │   ├── pages/                # Home, Dashboard, Games, Play, Leaderboard
│   │   └── context/              # Authentication & JWT bearer state
├── backend/                      # Express 5 + Mongoose Server (replaces artifacts/api-server)
│   ├── src/
│   │   ├── routes/               # Express endpoints (auth, users, sessions, leaderboards)
│   │   └── models/               # MongoDB Mongoose models (User, Session, LetterStat)
├── lib/
│   ├── api-spec/                 # OpenAPI 3.1 YAML specifications (Single Source of Truth)
│   ├── api-client-react/         # Generated React Query hooks (Orval integration)
│   └── db/                       # Migration legacy folder
└── package.json                  # Workspace build, typecheck, and dev scripts
```

### Stack Details
- **Frontend Framework:** React 19 (custom state tracking, React Query query management)
- **Styling:** Vanilla CSS variables + Tailwind CSS v4 for hardware-accelerated animations
- **Animations:** Framer Motion (page transitions, screen shakes, combat squash/stretch)
- **Database:** MongoDB Atlas via Mongoose ORM
- **API Spec:** OpenAPI 3.1 contract-first routing

## 🚀 Getting Started

### Prerequisites
- Node.js 20 or later
- pnpm 9 or later
- MongoDB Atlas cluster URI (local or cloud-hosted instance)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/typeblitz.git
   cd typeblitz
   ```

2. **Install all workspace packages:**
   ```bash
   pnpm install
   ```

3. **Configure environment variables:**
   Create a `.env` file at the root:
   ```env
   MONGODB_URI=mongodb+srv://<username>:<password>@cluster0.mongodb.net/typeblitz?retryWrites=true&w=majority
   SESSION_SECRET=a-long-random-string-used-for-auth-bearer-tokens
   ```

4. **Run the API server (Terminal 1):**
   ```bash
   pnpm --filter @workspace/api-server run dev
   ```

5. **Run the frontend (Terminal 2):**
   ```bash
   pnpm --filter @workspace/typeblitz run dev
   ```

## 📡 API Reference

Base URL: `/api`

### Authentication
- `POST /auth/register` - Create account `{ username, password }`
- `POST /auth/login` - Authenticate & obtain Bearer token `{ username, password }`
- `GET /auth/me` - Profile context for authenticated user

### Sessions & Performance
- `POST /sessions` - Save a finished typing session
- `GET /users/:userId/stats` - Fetch aggregate WPM, streak, and Global Rank
- `GET /users/:userId/letter-accuracy` - Get character heatmap error breakdown
- `GET /leaderboard` - Retrieve top rankings (filters out accuracy < 90%)

## 📐 Typing Speed & Accuracy Verification

TypeBlitz uses the official **international typing speed standards**:

### Words Per Minute (WPM) Formula
$$WPM = \frac{\text{Correct Characters} / 5}{\text{Minutes Elapsed}}$$

- Mistyped characters do **not** contribute to character counts for WPM calculation.
- WPM updates in real time on the UI every 800ms.
- Timers start on the very first character pressed, eliminating load delays.

### Accuracy Formula
$$\text{Accuracy (\%)} = \frac{\text{Correct Characters}}{\text{Total Typed Characters} + \text{Blocked Keystrokes}} \times 100$$
